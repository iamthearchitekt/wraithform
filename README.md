# WraithForm

A high-performance audio visualization tool featuring a custom OpenGL rendering engine.

## Features
- **Oscilloscope**: Real-time time-domain waveform visualization.
- **Spectrogram**: Detailed frequency analysis with heat-mapping.
- **Phase Meter**: Goniometer for stereo field analysis (Lissajous figures).
- **Serato-Style**: Scrolling multi-band waveform view.
- **Circular View**: Radial oscilloscope.
- **Quad View**: Simultaneous display of all major visualizers.
- **"Ice Blue" Theme**: Consistent, polished aesthetic (#D5FFFF).

## Installation
Currently available as a standalone Windows application (`.exe`).

## Build Instructions
1.  Clone the repository.
2.  Configure with CMake.
3.  Build using Visual Studio 2022 (or preferred generator).

## Tech Stack
- **JUCE Framework**: Audio and GUI foundation.
- **OpenGL**: Custom shader-based rendering.
- **Modern C++**: C++17/20 standards.
